# Nigeria Security Dashboard

## Overview
This project is a **prototype situational awareness dashboard** for Nigeria.  
It demonstrates how security locations can be visualized on a map with images and descriptions.

⚠️ **Disclaimer**
This application uses **simulated/demo data only**.  
It does **not** access real military satellites, surveillance systems, or classified intelligence.

## Features
- Interactive map of Nigeria
- Security markers with image popups
- Clean, mobile-friendly interface
- Ready for future authorized integrations

## Technology
- HTML, CSS, JavaScript
- Leaflet.js for mapping
- OpenStreetMap tiles

## Future Upgrades (Authorized Only)
- Satellite imagery feeds
- Drone & ISR data
- AI-based threat analysis
- Secure military backend systems

## Purpose
Educational, research, and prototype development.
